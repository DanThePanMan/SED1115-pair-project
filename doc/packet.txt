Packet general format:

┌──────┬───┬────┬──────┬──────┬──────┐
│ 0x47 │ 1 │ 11 │ .... | 0x30 | 0x48 |
└──────┴───┴────┴──────┴──────┴──────┘

[ 00..08 ]: Frame header = 0x47 (8 bits)
[ 08..10 ]: Packet type (2 bits) 
    - 0b00 -> ACK
    - 0b01 -> NACK
    - 0b1X -> Data
[ 10..11 ]: Sequence bit (1 bit)
[ 11..-16]: Data (variable size)
[-16..-8 ]: Checksum (8 bits)
[-08..00 ]: Frame trailer = 0x48 (8 bits)

(format is presented after decoding)
